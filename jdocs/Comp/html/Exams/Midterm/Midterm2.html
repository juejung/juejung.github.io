Midterm Exam Econ 431¶

Instructions¶

  • Read the instructions for each question very, very carefully.
  • Submit this exam via Dropbox just like you submitted homework assignments. There is already a folder with the midterm.py file inside your shared Dropbox folder. Due date: Wednesday, March 2nd, 11:55pm
  • I need to be able to quickly run these scripts on my computer. Make sure that your script runs through without any errors!! If the code breaks, you lose points.
  • This is an open book exam. You can use any material that you are able to find online.
  • You are not allowed to cooperate with anybody!!
  • If you have questions, contact me directly.

Question 1 (20 points)¶

Define vector \(x=[3.2, 6, 7.8, 1, 3, 2.5, 100]\).

  • Write a loop that adds the number 5 to all values of this vector that are smaller than 4. Do not just add this by hand, I need to see whether you can write a loop. You probably want to use an if statement as well.
  • Reassign the original values to vector x and write a loop that replaces all values between 2 and 5 with -1.
  • Reassign the original values to vector x. Write a loop and calculate the squared deviations of each value from the mean of all the values and store these squared deviations in a new vector called \(sqv\). In other words, calculate the average over all the numbers and then subtract this average from all the numbers etc.

Question 2 (20 points)¶

Calculate the sample covariance between two variables. Here are the values of the two variables:

\[x = [1., 5.5, 7.2, 4.2, -2.7, -5.4, 8.9]\]\[y = [0.1, 1.5, 1.8, -4.2, 2.7, -9.4, -1.9]\]

Hint: Define two vectors xv and yv using the values above. Calculate the average for each variable. Then write a loop that implements the formula for the sample covariance:

\[Cov(x,y) = \frac{\sum_{i=1}^{n}(x_i-\mu_x)(y_i-\mu_y)}{n-1}.\]

In order to get full points you really need to program this function by hand. Do not call a covariance routine from pandas or numpy but implement the above formula yourself.

Question 3 (20 points)¶

\[\begin{split}f(x) = \left\{ \begin{array}{ll} log(|x|) & \mbox{if } x < 0 \\ -x & \mbox{if } 0 \leq x < 2 \\ x^2/(3-x) & \mbox{if } 2 \leq x \end{array} \right.\end{split}\]

Plot this function \(-4 \leq x \leq 4\).

  • Hint: First define the gridvalues for \(x\) and store them in xv.
  • Then define a zero-vector yv of equal length.
  • Then write a loop over all the values of xv and calculate the corresponding values for yv and store them in the yv vector.
  • You need a series of if commands inside of this loop to distinguish the various cutoffs of the function.

Question 4: (20 points)¶

Define a matrix:

\[\begin{split}A = \left( \begin{array}{ccc} 1.2 & 3.4 & 10.3 \\ 2 & 8 & 78 \\ 45 & -36 & 8 \end{array} \right).\end{split}\]

Write a loop and using if commands replace all the numbers in the main diagonal (top left to bottom right diagonal) with -5. You need to write a loop to replace the numbers in the main diagonal. You need to demonstrate how to loop over an array and how to use if - statements! I will not give you points if you simply type the solution values:

\[\begin{split}A = \left( \begin{array}{ccc} -5 & 3.4 & 10.3 \\ 2 & -5& 78 \\ 45 & -36 & -5 \end{array}\right).\end{split}\]

Next, write a loop and replace numbers in the off-diagonal with 100. Again, use a loop and if commands in order to get full points, simply typing the new matrix:

\[\begin{split}A = \left( \begin{array}{ccc} 1.2 & 100 & 100 \\ 100 & 8 & 100 \\ 100 & 100 & 8 \end{array} \right)\end{split}\]

will not result in any points.

Question 5 (20 points):¶

You have information about the following six people:

Name Marriage Status Income Spouse
Julie married $35,000 Jack
Angie not married $55,000 na
Sarah married $45,000 Jim
Jack married $35,000 Sarah
John not married $25,000 na
Jim married $35,000 Julie

Define a nested list (you need to just type this) that contains the entire information of this table as follows:

personList = [['Julie', 'married', 35000, 'Jack'], \
              ['Angie', 'not married', 55000, 'na'], etc.]
  • Write a loop and create a new list called marriedList that only contains individuals that are married. In order to get full points you need to write a loop that steps through the list, checks the marriage status of each individual and then grabs the information of the married ones and puts them into the new list. This new nested list should only contain info about:
Name Marriage Status Income Spouse
Julie married $35,000 Jack
Sarah married $45,000 Jim
Jack married $35,000 Sarah
Jim married $35,000 Julie
  • Write a loop that steps through the personList and calculates the total income of each household.

The printout should look like this:

Household                  Total Income
----------------------------------------
1                           70000
2                           55000
3                           80000
4                           25000

Explanation: Household 1 is Julie and Jack, household 2 is just Angie, household 3 is Sarah and Jim and household 4 is just John.

Hint: This requires a lot of list manipulation and list indexation. There are commands that allow you to append to lists and there are commands that allow you to delete elements from a list. A combination of for loops, possible nested, and some if commands should do the trick. Give it a try.
